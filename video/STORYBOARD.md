## The real game (so footage stays true to it)
Vault Run is a **live crash/heist multiplier game**: a vault dial counts a
multiplier up from **1.00x**; the **crowd votes live** on actions —
**HACK LASERS / DISABLE DRONE / OPEN BONUS VAULT / RUSH EXIT**; a security-threat
meter rises; players **cash out** before it busts; a **Top Heists** leaderboard
shows monster wins (128.4x, 92.1x…). The creator runs it; the audience controls it.
